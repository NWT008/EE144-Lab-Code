# EE144-Lab-Code
Course Lab work for EE144: Foundations of Robotics

The format of this these programs were to fill in the TODO sections utilizing given equations to create a working UKF, EKF, particle filter and more. Each folder covers a different filter as well as collision models, robot simulation, and sensor/beacon localization. There are scrpits that have been omitted because I no longer have access to them. However, these were the core assignments that display the behavior of the stated filter and a report that noted the results of each simulation.
